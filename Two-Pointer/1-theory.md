# Two pointer:
the two pointer approach is a problem-solving technique where you use two indices (pointers) <br>
to traverse a data structure (usually an array or string) in a smart way instead of using nested loops.
<br>
                                                         
It helps reduce time complexity, often from O(n²) → O(n).

## Basic Idea:

Instead of checking every possible pair using two loops Use two pointers <br>
-Move them based on conditions
<br>
-Stop when the goal is achieved
